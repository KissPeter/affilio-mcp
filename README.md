# Affilio MCP Link Shortener

Free URL shortener MCP tool powered by Affilio.link — 
the smart affiliate link management platform for content creators.

## What it does
- Shortens any URL to a branded mcp.affilio.link short URL
- Returns a QR code for every link
- Links expire after 30 days
- No authentication required
